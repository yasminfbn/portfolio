MEU PROJETOS

Projeto Seleção da Copa 2026

Desenvolvimento de um sistema web para simular a seleção de países para a Copa do Mundo de 2026, utilizando HTML, CSS, JavaScript e PHP com integração a banco de dados MySQL.

O projeto implementa um CRUD completo (criar, ler, atualizar e excluir) para o gerenciamento das seleções, permitindo que o administrador cadastre, visualize, edite e remova equipes.

Foram aplicadas práticas de segurança como validação de dados e uso de prepared statements para evitar SQL Injection, além de impedir duplicidade de seleções.

HTML CSS JavaScript PHP MySQL

Chat ao Vivo com Sistema de Login

Desenvolvimento de um chat ao vivo com sistema de login completo, utilizando PHP, JavaScript, HTML e CSS, integrado a um banco de dados MySQL.

A aplicação permite cadastro, login e envio de mensagens em tempo real.

As senhas são protegidas com password_hash() e password_verify(), além de validação e sanitização de dados e filtro de palavras ofensivas.

HTML CSS JavaScript PHP MySQL

Estudo sobre API: Previsão do Tempo em Python

Desenvolvimento de um programa em Python para consumo de API de clima, exibindo dados meteorológicos em tempo real.

O sistema processa e organiza as informações de forma clara para o usuário.

Foi utilizado arquivo .env para proteger a chave da API, garantindo segurança das credenciais e boas práticas no uso de APIs.
